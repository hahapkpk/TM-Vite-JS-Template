主要功能发布

2025年5月22日
Claude Sonnet 4 集成
重大发布 性能提升

Augment Code 向所有用户推出了Claude Sonnet 4，这是Anthropic最新且最强大的编程模型。这代表了AI能力的重大升级：

  • SWE-bench 性能：单次通过评分从60.6%提升至70.6%
  • 回归测试套件：通过率从46.9%提升至63.1%（+34.5%）
  • 工具调用准确性：有效工具调用率从25.0%跃升至80.0%（+220%）
  • 编辑精度：限制内编辑率从21.4%提升至64.3%（+200.5%）

升级保持相同价格，同时提供显著更好的代码生成和编辑能力。

2025年6月5日
Remote Agents 正式发布
重大发布 生产力提升

Remote Agents 在VS Code中正式发布，使工程师能够在云端并发运行多个开发任务：

  • 自主操作：代理在您注销后继续工作
  • 并行处理：同时运行多达10个代理
  • 企业级隐私：不可提取架构，严格的无训练保证
  • 智能上下文：语义索引在毫秒内检索相关代码

使用场景包括处理技术债务、重构代码、提升测试覆盖率和并行探索多种解决方案。

2025年7月8日
Augment Rules 系统
开发者体验

引入了Augment Rules - 指导AI代理行为的指令文件，提供精细控制：

  • 灵活组织：在.augment/rules文件夹中创建规则文件
  • 三种使用模式：始终、手动或自动（智能规则选择）
  • 轻松迁移：自动从竞争对手的规则文件夹导入
  • 智能选择：代理自动确定相关规则

规则帮助代理理解编码标准、项目要求和特定工作流程。

性能与技术改进

2025年6月11日
使用量化向量搜索实现40%更快的代码搜索
性能提升

使用量化向量搜索对大型代码库进行重大性能优化：

  • 内存减少：8倍减少（1亿行代码库从2GB减少到250MB）
  • 延迟改进：搜索时间从2+秒减少到200毫秒以下
  • 准确性保持：对精确结果保持99.9%的保真度
  • 可扩展性：无缝支持多达1亿+行代码的代码库

优化使用近似最近邻（ANN）算法，对边缘情况有自动回退。
